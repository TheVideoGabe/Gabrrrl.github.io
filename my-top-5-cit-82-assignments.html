My Top 5 CIT-82 Assignments

Here are some of my favorite assignments.

Posted by Gabriel C. on September 24, 2021

GitHub Pages

Learning about github, github pages and how to keep track of our files by using our github account was a great assignment and learning experience. I think using GitHub is a very fundamental skill we all should have.

VS Code And Git

When we learned about VS Code i was intimidated by an IDE especially when we couldn't use github to change the code but after learning how to use git, and becoming used to the git workflow it made me fall in love with the process of pushing code into our repos.

Bootstrap And Templates

I always tried to set up a boot strap theme on github pages trough a tutorial but I didn't have that git workflow core skill so I was never able to launch a succesful theme on my website. Thanks to this assignment I feel more comfortable not only installing themes for my website but also other GitHub open source projects.

Learning About SEO

This assignment taught me a lot about SEO and how to take advantage of search engines for my own websites. I had seen videos on how to boost your websites ratings by using keywords inside of your pages, another important thing I took away from this assignment is how to create content people will look for online, and tips and tricks on how to do that.

Building Our Pages

I have been having a blast creating pages for my website. This assignment is giving us practice on how to build websites and giving us the freedom to create anything we want.
